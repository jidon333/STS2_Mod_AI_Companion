# ROADMAP

## Phase 1

Separate-process coaching assistant that observes exported game state and provides out-of-process guidance.

## Phase 2

Bridge design between the native STS2 mod and the external companion host, with clear contracts for state snapshots and commands.

## Phase 3

In-game UI or overlay integration that can surface companion output without committing to intrusive gameplay hooks.

## Phase 4

Feasibility study for a multiplayer AI teammate, including technical limits, fairness concerns, and rollback options.
