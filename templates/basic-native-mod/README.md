# basic-native-mod

These files are the Godot export-pack template assets used by `build-native-pck`.

They are intentionally generic and should stay stable unless the native package format changes.

Current repository baseline:

- runtime package name: `sts2-mod-ai-companion.pck`
- runtime config sample: `sts2-mod-ai-companion.config.sample.json`
- managed project: `src\Sts2ModAiCompanion.Mod`
